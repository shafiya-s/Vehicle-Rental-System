# 🚗 Vehicle Rental System

A simple **Java Console-Based Vehicle Rental System** developed using **Object-Oriented Programming (OOP)** concepts. This application allows users to view available vehicles, rent a vehicle, and calculate the rental bill through a menu-driven interface.

## 📌 Problem Statement

Many vehicle rental businesses manage vehicle details and rentals manually, which can lead to errors and consume more time. This project automates the rental process by allowing users to view vehicles, rent them, and generate bills using a simple Java console application.

## ✨ Features

- View Available Vehicles
- Rent a Vehicle
- Bill Calculation
- Menu-Driven Console Application
- Simple and Easy to Use

## 🛠️ Technologies Used

- Object-Oriented Programming (OOP)

## 🧩 OOP Concepts Used

- **Encapsulation** – Private variables with Getter methods
- **Inheritance** – `Car` and `Bike` inherit from `Vehicle`
- **Polymorphism** – Method overriding using `displayDetails()`
- **Constructors** – Object initialization using constructors

## ▶️ How to Run

1. Clone the repository.
2. Open the project in VS Code or Eclipse.
3. Compile all Java files.
4. Run `Main.java`.
5. Use the menu to view and rent vehicles.
